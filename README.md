# 实现功能
1.批量导入excel csv 数据进入mysql 数据库。
2.文件检索，可以上传单个excel 或者csv文件，也可以将多个csv或者excel打成一个压缩包zip 进行上传。rar目前还在开发。
3.添加数据库功能。并能管理数据库连接。
4.实现了一个简单的数据库管理工具，可以进行sql 语句的执行。
5.实现了初级数据分析，银行流水分析。
